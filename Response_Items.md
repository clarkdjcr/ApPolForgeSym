#Response Process
1. Read rejection message carefully
2. Note all issues mentioned
3. Fix issues in code/metadata
4. Create new build if code changes needed
5. Update metadata if needed
6. Write response to reviewer
7. Resubmit

## Quick Reference

### Required Account Types
- **Apple ID:** Free
- **Apple Developer Program:** $99/year (required for App Store)

### Required Metadata
- App name (30 chars)
- Subtitle (30 chars)
- Description (4000 chars)
- Keywords (100 chars)
- Screenshots (5-10 per device size)
- App icon (1024x1024)
- Privacy policy URL
- Support URL

### Required Testing
- Physical device testing
- iOS minimum version (17.0)
- iOS latest version
- TestFlight beta (recommended)

### Timeline Estimate
- Preparation: 1-2 weeks
- TestFlight beta: 1-2 weeks
- App review: 1-3 days
- **Total: 2-5 weeks** from start to App Store
