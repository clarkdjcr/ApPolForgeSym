## Third-Party Services

### Optional AI Integration (OpenAI / Anthropic Claude)

ApPolForgeSym includes **optional** integration with external AI services for enhanced strategic recommendations. This feature is:

- ‚ö†Ô∏è **Entirely optional** - Not required to play the game
- ‚ö†Ô∏è **User-initiated** - Only activated if you add your own API key
- ‚ö†Ô∏è **Your account** - Uses your personal OpenAI/Anthropic account
- ‚ö†Ô∏è **Your responsibility** - Subject to those services' privacy policies

**How It Works:**
1. You create an account with OpenAI or Anthropic (optional)
2. You obtain an API key from their service
3. You enter that API key in ApPolForgeSym settings
4. API key is stored securely in iOS Keychain (encrypted, device-only)
5. When you request AI advice, the app sends game state data to the AI service
6. The AI service processes your request and returns recommendations
7. You are billed by OpenAI/Anthropic according to their pricing

**Data Shared with AI Services (when you use this feature):**
- Game state (current week, polling numbers, funds, state data)
- Your campaign actions and history
- Nothing personally identifiable (no name, email, device ID)

**Important Notes:**
- We don't receive, store, or have access to this data
- The AI service may log requests per their privacy policy
- Review [OpenAI Privacy Policy](https://openai.com/privacy) and [Anthropic Privacy Policy](https://www.anthropic.com/privacy)
- You can remove API keys at any time
- Game fully functional without this feature
